## Local Setup

### 1. Clone the repo 
```bash
git clone https://github.com/irahuldutta02/simple-react-todo-app.git
cd simple-react-todo-app
```
### 2. Install dependencies

```bash
npm install
```
### 3. Start the development server

```bash
npm run dev
```
